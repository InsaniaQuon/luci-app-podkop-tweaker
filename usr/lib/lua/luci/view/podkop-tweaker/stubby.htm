Fix Init Script

        

            
Problem:

            
The init script /etc/init.d/stubby runs the process as user stubby, which lacks write permissions for /var/lib/stubby/ and cannot bind to privileged port 53.

            
One of the solutions:

            
Replace the user with root in the init script:

            
sed -i 's/procd_set_param user stubby/procd_set_param user root/' /etc/init.d/stubby
